## ðŸš€ Quick Start Guide

### Logging In

1. Navigate to the login page
2. Enter your username and password
3. Click "Sign In"

### Creating Your First Ticket

1. Click "Create Ticket" from the Dashboard or Tickets menu
2. Select the Site and Asset (optional)
3. Choose Category and Priority
4. Enter Title and Description
5. Click "Submit"

### Navigating the System

- **Sidebar**: Access all modules from the left navigation
- **Dashboard**: View real-time statistics and quick actions
- **Search**: Use the global search to find tickets, assets, or users
