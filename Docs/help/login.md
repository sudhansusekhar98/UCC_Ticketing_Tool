# Login & Authentication

## Overview

The Login page is your gateway to the TicketOps system. Secure authentication ensures only authorized users can access the platform.

---

## Logging In

### Steps:

1. Navigate to the TicketOps URL
2. Enter your **Username**
3. Enter your **Password**
4. Click **"Sign In"**

### Successful Login:

- You'll be redirected to Dashboard
- Your session is saved
- You remain logged in until timeout

### Failed Login:

- Error message displayed
- Check username/password
- Contact admin if locked out

## Forgot Password

### If You Forgot Your Password:

1. Contact your System Administrator
2. Admin will reset your password
3. You'll receive new temporary password
4. Login and change immediately

### Self-Service (if enabled):

1. Click "Forgot Password?"
2. Enter your email
3. Check email for reset link
4. Click link and set new password

---

## Account Locked

### Why Accounts Get Locked:

- Too many failed login attempts
- Admin manually locked account
- Security policy violation

### Unlock Account:

1. Wait for lockout period (usually 15 min)
2. Or contact administrator
3. Admin can unlock immediately

---

## First Time Login

### New User:

1. Receive credentials from admin
2. Login with provided username/password
3. You may be prompted to change password
4. Set a new secure password
5. Complete your profile

### Password Requirements:

- Minimum 8 characters
- Mix of uppercase and lowercase
- Include numbers
- Include special characters

---

## Multi-Device Access

### Login from Multiple Devices:

- You can login from multiple browsers
- Each creates a new session
- All sessions share same account
- Actions sync across devices

### Managing Sessions:

- Go to Profile > Active Sessions
- View all logged-in devices
- Sign out from specific devices
- Sign out all devices if needed

---

## Session Timeout

### Automatic Logout:

- Sessions expire after inactivity
- Default: 30 minutes
- Warning shown before timeout
- Click "Stay Logged In" to extend

### After Timeout:

1. You're logged out
2. Redirected to login page
3. Login again to continue
4. Unsaved work may be lost

### Preventing Timeout:

- Keep the browser tab active
- Interact with the system periodically
- Save work frequently

---

## Logout

### How to Logout:

1. Click your avatar (top-right)
2. Select "Logout"
3. Confirm if prompted
4. You're logged out

### Why Logout:

- End of work session
- Shared computer
- Security best practice
- Switching accounts

---

## Troubleshooting

### Can't Login:

| Issue            | Solution                           |
| ---------------- | ---------------------------------- |
| Wrong password   | Check caps lock, try again         |
| Account locked   | Wait or contact admin              |
| Forgot password  | Contact admin for reset            |
| Page not loading | Check internet, try refresh        |
| "Server error"   | Wait and retry, or contact support |

### Browser Issues:

- Clear browser cache
- Disable ad blockers
- Try incognito/private mode
- Try different browser

### Still Can't Login:

1. Note the exact error message
2. Contact system administrator
3. Provide your username
4. Don't share your password

---

## Security Tips

1. **Never share credentials** - Keep password secret
2. **Use strong passwords** - Mix characters and numbers
3. **Logout when done** - Especially on shared devices
4. **Report suspicious activity** - If you notice anything odd
5. **Don't save on public computers** - Avoid "remember me"
