## Accessing Your Profile

### Methods:

1. Click on your avatar in the top-right corner
2. Select "Profile" from dropdown
3. Or navigate via sidebar (if available)

---

## Editing Profile

### Steps:

1. Go to Profile page
2. Click "Edit Profile" button
3. Modify allowed fields:
   - Full Name
   - Email
   - Phone
4. Click "Save Changes"

### Profile Picture:

1. Click on avatar image
2. Select "Change Photo"
3. Upload new image (JPG/PNG)
4. Crop if needed
5. Save

#### Image Requirements:

- Format: JPG, PNG
- Max Size: 2MB
- Recommended: 200x200px

---

## Changing Password

### Steps:

1. Go to Profile page
2. Click "Change Password"
3. Enter Current Password
4. Enter New Password
5. Confirm New Password
6. Click "Update Password"

### Password Requirements:

- Minimum 8 characters
- At least 1 uppercase letter (A-Z)
- At least 1 lowercase letter (a-z)
- At least 1 number (0-9)
- At least 1 special character (!@#$%^&\*)

### Password Tips:

- Use a unique password
- Don't reuse old passwords
- Use a password manager
- Change regularly (every 90 days)

## Notification Preferences

### Configure Notifications:

1. Go to Profile
2. Click "Notification Settings"
3. Toggle options:

#### In-App:

- [ ] Ticket Assignments
- [ ] Status Changes
- [ ] Comments
- [ ] SLA Warnings
- [ ] RMA Updates

#### Email:

- [ ] New Assignments
- [ ] SLA Alerts
- [ ] Daily Digest
- [ ] Weekly Summary

4. Save preferences

## Session Management

### Active Sessions:

View all your active login sessions:

- Current session (highlighted)
- Other devices
- Login time
- IP address
- Browser/Device

### Terminate Session:

1. Find session to end
2. Click "Sign Out"
3. Confirm action

### Sign Out All:

1. Click "Sign Out All Devices"
2. Confirm
3. You'll be logged out too
